IComponentSystem* createComponentSystem ( )

Factory function for creating an instance of the draw component system.

Returns
A pointer to a new instance of the draw component system.

The function is declared as extern "C", which specifies that it has C linkage (i.e., it can be called from C code).

This is necessary because the component system is loaded as a shared library.

System responsible for updating and drawing rectangular shapes.

Parameters
regA reference to the entity-component system registry.
rectsA sparse array of components that contain the rectangular shape properties for each entity.
imagesA sparse array of components that contain the image properties for each entity.
drawA sparse array of components that determine whether an entity should be drawn.

This system iterates through the sparse arrays of components (images, rects, and draw)

and for each entity with all three components, it checks if the image has a valid

database pointer. If so, it checks if the entity has a valid sf::IntRect object in

the database. If not, it creates one, otherwise it updates and loads the existing one.
