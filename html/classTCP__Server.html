void handle_client(tcp::socket socket)

This function will launch the tcp server, it will create one thread, a thread that has the function handling the clients.

It will create a socket and an acceptor, the acceptor will accept the connection from the client and the socket will be used to send and receive data from the client

The thread will be detached so it will be able to run in the background and the main thread will be able to continue.
