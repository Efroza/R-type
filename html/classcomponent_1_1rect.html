Parameters
lineThe number of lines.
colsThe number of columns.

The entity will be associated with a given rect or not depending on the number of lines and columns.

If the number of lines and columns is 0, the entity will not be associated with a rect.

If the number of lines and columns is not 0, the entity will be associated with a rect to be able to possibibly get an animation or other actions on it.
